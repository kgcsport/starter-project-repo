# Starter template for futurew projects

If I use this template, I will get several folders necessary for my research project workflow. These are:

- code
  - code/build
  - code/analysis
- data
  - data/raw
  - data/work
- output
  - output/figures
  - output/tables
- documentation
- presentations
- writing
- literature

Also, it creates a housekeeping.R file.
